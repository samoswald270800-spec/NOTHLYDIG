# Northly Digital — Premier Staffing & Talent Solutions

A premium, marketing-grade landing page for **Northly Digital**, a Canadian staffing
company connecting world-class talent with ambitious organizations.

Built with HTML, CSS, and vanilla JavaScript — no frameworks, no build step.

## Highlights
- Aurora-inspired premium dark theme (mint · sky · violet gradient palette)
- Sticky glass navbar with scroll-aware blur and active-link tracking
- Hero with animated word reveal, parallax auroras, and trust marquee
- Animated stat counters with `IntersectionObserver`
- Cursor-aware service cards with gradient-mask borders
- Magnetic CTAs with subtle pull on pointer hover
- Fully responsive — mobile hamburger, fluid grids
- Honors `prefers-reduced-motion`
- Contact CTA: **+1 (613) 817-4782** · `hello@northlydigital.com`
