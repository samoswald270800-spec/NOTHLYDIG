# Apex Talent Group — Staffing Agency Website

A modern, responsive staffing agency landing page built with HTML, CSS, and vanilla JavaScript.

## Features
- Fixed glassmorphism navbar with scroll effect
- Animated hero with floating orbs and grid overlay
- Animated stat counters (IntersectionObserver)
- Services, Industries, Process, Testimonials, and Contact sections
- Working contact form with simulated submission
- Scroll-reveal animations on cards
- Fully responsive (mobile hamburger menu)
- Dark-mode premium design with gradient accents

## Stack
- HTML5
- Vanilla CSS (custom design system, no frameworks)
- Vanilla JavaScript (no dependencies)

## Getting Started
Just open `index.html` in a browser — no build step required.
